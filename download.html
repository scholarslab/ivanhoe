Download and Install

To play Ivanhoe, you will first need to install WordPress and set up your own WP page. Once you are set up on WordPress, click the button below to download our Ivanhoe Game WP Theme.

Download Version 1.1!

After downloading the .zip file for the Ivanhoe Game WP Theme, follow WP's instructions on adding new themes manually. For instructions on playing Ivanhoe, see our Documentation.

As Ivanhoe is a work in progress, we would love your feedback! Feel free to post any issues you find to our GitHub page.
