Style Guide

This document contains a sample of HTML elements that can exist on the site, to ensure proper styling.

Block Level Elements

Headings

Headings are use to mark sections of content. There are six levels of headings.

First-Level Heading

A first-level heading is used for the main header of a page. Only one should be used.

Second-Level Heading

A second-level heading should come after a first-level heading in the document hierarchy. More than one can be used.

Third-Level Heading

A third-level heading should come after a second-level heading in the document hierarchy. More than one can be used.

Fourth-Level Heading

A fourth-level heading should come after a third-level heading in the document hierarchy. More than one can be used.

Fifth-Level Heading

A fifth-level heading should come after a fourth-level heading in the document hierarchy. More than one can be used.

Sixth-Level Heading

A sixth-level heading should come after a fifth-level heading in the document hierarchy. More than one can be used.

Paragraphs

Paragraphs are wrapped with a <p> tag. They can contain a variety of other inline elements, such as links, emphasized text, strong text, code, deleted text, and inserted text

Blockquotes

Blockquotes are wrapped with a <blockquote> tag, and can contain a variety of other markup, including paragraphs, headings, and lists.

Nulla vitae elit libero, a pharetra augue. Fusce dapibus, tellus ac cursus commodo, tortor mauris condimentum nibh, ut fermentum massa justo sit amet risus. Cras justo odio, dapibus ac facilisis in, egestas eget quam.

Tables

A table is used to display tabular data. It uses a <table> tag, and requires several descendant tags to denote rows (<tr>) and divisions within rows (<td>). Tables can have a head section using the <thead> tag, and a footer using the <tfoot> tag.
